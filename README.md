# Luxury Hotel Supplies Project

A premium B2B Next.js 14 web application tailored for luxury hotel procurement in the UAE.

## Built With

- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS, Material UI (MUI)
- **Design System**: Custom CSS with "Gold/Ink" luxury aesthetic

## Features

- High-end masonry grids for product showcases
- Custom interactive UI components with glassmorphism and animations
- Dark-mode luxury aesthetics
- Fully responsive design

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
